# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer built using **Streamlit** and **Groq LLM**. It analyzes resumes, identifies strengths and weaknesses, and provides actionable suggestions for improvement.
## Live Demo : 

🌐 https://ai-resume-critiquer-analyzer-fytxkg3gzueffabjut5sbf.streamlit.app/

---

## ✨ Features

*  Upload PDF resumes
*  AI-powered resume analysis
*  Skill assessment
*  Resume improvement suggestions
*  ATS-friendly feedback
*  Fast analysis using Groq LLM
*  Simple and interactive    
*  Streamlit interface

---

## ⚙️ How It Works

* 1️⃣ Upload your resume

* 3️⃣ Send data to the AI model

## 📦 Installation

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/resumeanalyzer.git 
cd resumeanalyzer 
```

### 2. Create a Virtual Environment

```bash
python -m venv .venv
```

### 3. Activate the Environment:
**Windows**:

```bash
.venv\Scripts\activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Configure API Key

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key
```

### 6. Run the Application

```bash
streamlit run main.py
```
